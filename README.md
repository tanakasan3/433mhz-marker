# 433MHz Marker

OpenSCAD model for a small marker/label piece.

## Specifications

- **Dimensions:** 8mm × 8mm × 10mm (square prism)
- **Center hole:** 4mm diameter (through entire height)
- **Text:** "433MHz" engraved on all 4 faces, oriented vertically

## Usage

Open `433mhz_marker.scad` in OpenSCAD and render (F6) to generate the model. Export as STL for 3D printing.

## Customization

Edit the parameters at the top of the file:
- `side` - Square side length (default: 8mm)
- `height` - Total height (default: 10mm)
- `hole_diameter` - Center hole diameter (default: 4mm)
- `text_depth` - Engraving depth (default: 0.5mm)
- `text_size` - Font size (default: 2)
